# Lightning Agent Tools

Lightning Agent Tools packages an MCP server, a guarded node-ops daemon, and
agent skills for operating Lightning Network development and regtest
environments. The documentation site collects the repository's canonical docs
and exposes LLM-friendly indexes for agents that need concise source context.

## Start Here

- Start with Getting Started for install choices, the first regtest run, and
  the guarded node-ops workflow.
- Use Quick Reference when you need the command surface in one page.
- Use Skill Reference for generated pages from every `skills/*/SKILL.md`.
- Use `llms.txt` for an AI-friendly documentation index.
- Use `llms-full.txt` when an agent needs the full generated docs bundle in one
  file.

## Documentation Sources

This site is generated from repository sources: canonical pages in `docs/`,
agent guidance in `AGENTS.md`, Docusaurus pages in `docs-site/docs/`, and skill
reference pages generated from `skills/*/SKILL.md`.
